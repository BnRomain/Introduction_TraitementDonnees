# Dictionnaire des données : Himalayan Expeditions

| Variable | Description |
| :--- | :--- |
| `year` | Year of the expedition |
| `season` | Season in which the expedition took place (Spring, Summer, Autumn, Winter) |
| `peakid` | ID of the peak being climbed |
| `totmembers` | Total number of team members in the expedition |
| `smtmembers` | Number of members who successfully summited |
| `mdeaths` | Number of members who died during the expedition |
| `tothired` | Total number of hired staff (e.g., Sherpas) |
| `smthired` | Number of hired staff who summited |
| `hdeaths` | Number of hired staff who died |
| `comrte` | Whether the expedition used a commercial route (boolean) |
| `o2used` | Whether bottled oxygen was used during the expedition |
| `bcdate` | Base camp arrival date |
| `smtdate` | Summit day date |
| `termdate` | Termination date |
| `totdays` | Total number of days spent on the expedition |
| `camps` | Number of high-altitude camps used |
| `rope` | Total length of fixed rope used during the climb |
| `host_country` | Host country from which the expedition began |
| `expedition_nation` | Main nationality of the expedition |
| `primary_route` | Name of the main climbing route |
| `success` | Whether the expedition successfully summited (binary target variable) |
